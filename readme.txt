== Description ==

Add cost price for products to workout profits per order. Every product in the order must have a cost price for it to be displayed. In order for this plugin to work all products must have a cost price. You can see which products are missing the cost price on the Products->All Products page in WooCommerce.

== Installation ==

1. Extract "profit-calculator-for-woocommerce.zip".
2. Upload "profit-calculator-for-woocommerce" folder via FTP to the WordPress directory: "/wp-content/plugins/".
3. Activate the plugin through the WordPress admin "Plugins" menu.
4. Go to each product and ensure all products have a cost price.
